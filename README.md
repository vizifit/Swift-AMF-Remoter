# Swift-AMF-Remoter

[![Swift][swift-badge]][swift-url]

**Swift-AMF-Remoter** provides an API for advanced reflection at runtime including dynamic construction of types.

## Usage

```swift
import Reflection

struct Person {
  var firstName: String
  var lastName: String
  var age: Int
}
